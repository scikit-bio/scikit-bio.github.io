skbio.core.ssw.AlignmentStructure.cigar¶

AlignmentStructure.cigar¶

Cigar formatted string for the optimal alignment

Returns:

str

The cigar string (described in [R22], [R23]) of the optimal alignment

Notes

If there is no cigar or optimal alignment, this will return an empty string

References

[R22](1, 2) http://genome.sph.umich.edu/wiki/SAM
[R23](1, 2) http://samtools.github.io/hts-specs/SAMv1.pdf
