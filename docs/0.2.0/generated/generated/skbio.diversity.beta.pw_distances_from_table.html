skbio.diversity.beta.pw_distances_from_table¶

skbio.diversity.beta.pw_distances_from_table(table, metric='braycurtis')[source]¶

Compute distances between all pairs of samples in table

Parameters:

table : biom.table.Table

Table containing count/abundance data of observations across samples.

metric : str, optional

The name of the pairwise distance function to use when generating pairwise distances. See the scipy pdist docs, linked under See Also, for available metrics.
Returns:

skbio.DistanceMatrix

Distances between all pairs of samples. The number of row and columns will be equal to the number of samples in table.

See also

scipy.spatial.distance.pdist, biom.table.Table, pw_distances
