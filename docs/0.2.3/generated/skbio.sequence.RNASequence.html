skbio.sequence.RNASequence¶

class skbio.sequence.RNASequence(sequence, id='', description='', quality=None, validate=False)[source]¶

Base class for RNA sequences.

An RNASequence is a NucelotideSequence that is restricted to only containing characters used in the IUPAC RNA lexicon.

Notes

All uppercase and lowercase IUPAC RNA characters are supported.

Attributes

description Description of the biological sequence.
id ID of the biological sequence.
quality Quality scores of the characters in the biological sequence.
sequence String containing underlying biological sequence characters.
