DNA.__iter__()[source]#

Iterate over positions in this sequence.

Yields:
Sequence

Single character subsequence, one for each position in the sequence.

Examples

>>> from skbio import Sequence
>>> s = Sequence('GGUC')
>>> for c in s:
...     str(c)
'G'
'G'
'U'
'C'
