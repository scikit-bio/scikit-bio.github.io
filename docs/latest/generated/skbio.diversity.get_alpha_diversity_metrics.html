skbio.diversity.get_alpha_diversity_metrics()[source]#

List scikit-bio’s alpha diversity metrics.

The alpha diversity metrics listed here can be passed as metrics to skbio.diversity.alpha_diversity.

Returns:
list of str

Alphabetically sorted list of alpha diversity metrics implemented in scikit-bio.
